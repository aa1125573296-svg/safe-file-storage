## 🖥️ What this app does

safe-file-storage helps you store files in a secure way on your Windows PC. It uses AES-256 encryption to protect your files, scans uploads with VirusTotal, and keeps an audit log for admin review.

Use it when you want to:

- store private files in one place
- protect files with encryption
- check files for malware before use
- review file activity in an admin dashboard
- share files with more control

## 🔐 First-time setup

When you open safe-file-storage for the first time, follow the setup steps on screen.

You may be asked to:

- create an admin account
- set a password
- choose a storage folder
- connect the VirusTotal API key
- confirm encryption settings

Use a strong password that you do not use elsewhere.

If the app asks for a VirusTotal API key, paste it into the setup field and save it.

## 📂 How to store files

1. Open the app.
2. Sign in with your admin or user account.
3. Choose Upload or Add File.
4. Select the file from your computer.
5. Wait while the app scans the file.
6. Review the scan result.
7. Save the file to secure storage.

The app encrypts files with AES-256 before saving them.

## 🧪 Malware scanning

Before a file goes into storage, the app checks it with VirusTotal.

If the scan finds a risk, review the file before you open it.

If the file looks safe, you can continue with normal storage or sharing steps.

## 🧾 Audit log dashboard

The admin dashboard keeps track of file actions and account activity.

You can use it to review:

- uploads
- downloads
- sign-ins
- failed access attempts
- file changes
- scan results

This helps you see who did what and when.

## 🧰 Troubleshooting

### The app will not open
- Check that the file finished downloading
- Extract the ZIP file if needed
- Right-click the app and choose Run as administrator
- Try again after restarting Windows

### Windows blocks the app
- Select More info if Windows shows that option
- Choose Run anyway if you trust the release source
- Make sure you downloaded the latest version

### File scanning does not work
- Check your internet connection
- Confirm that your VirusTotal API key is correct
- Save the key again in the setup screen
- Try another file

### Stored files do not open
- Make sure you are signed in
- Check that you have permission for that file
- Confirm that the storage folder still exists
- Try restarting the app
